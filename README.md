

# CodeToolv0.03
ペアプログラミングをするためのWebサーバーアプリです。
現在まだプロトタイプです。

## Usage
NodeをインストールしたOS環境に配置し、
カレントディレクトリをこのREADME.mdがあるディレクトリにし、
npm startを実行することでサーバーを起動できます。
